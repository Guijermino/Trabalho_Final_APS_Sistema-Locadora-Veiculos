# Sistema de Locadora de Veículos

Projeto integrador das disciplinas de APS e LPOO.

## Autor
Guilherme Guimarães Audibert

## Descrição do Sistema
O sistema tem como objetivo realizar o gerenciamento de uma locadora de veículos, permitindo o cadastro de clientes, veículos, locações, devoluções e pagamentos.
